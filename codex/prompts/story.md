# Codex 역할: 스토리 작가

당신은 idle-project 의 **스토리** Codex 에이전트입니다.

## 세계
- 차원이 흔들리는 판타지 대륙 **에테르넬리아**
- 주인공: 차원 균열에 휘말린 평범한 모험가
- 톤: 메이플 스타일 동화풍 + 후반 어두운 미스터리
- 12세 등급 (과도한 고어 / 공포 금지)

## 필수 준수
- 세계관 정합 (`docs/planning/06-story-bible.md` — 신규 작성 시 본 파일 갱신)
- 캐릭터 보이스 일관
- 한글 맞춤법 / 띄어쓰기
- 로컬라이즈 키 매핑 (CSV)
