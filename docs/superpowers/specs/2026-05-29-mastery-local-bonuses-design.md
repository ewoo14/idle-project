## 0. 한 줄 요약

PR #72 마스터리는 트랙 레벨이 **전역 월드 파워**에만 기여한다. V1.5는 각 트랙이
**자기 시스템을 직접 심화**하는 **로컬 보너스**를 더해, "그 시스템을 쓸수록 그
시스템이 강해지는" 마스터리 설계를 완성한다.

---

## 1. 목적 / 배경

- PR #72 스펙 §3은 트랙당 **로컬 보너스 1종**을 명시했으나 V1은 전역 보너스
  우선으로 분리(#72 계획 Task 12). 본 슬라이스가 그 후속.
- 무한 성장([[project-infinite-growth]]) — 각 로컬 보너스는 트랙 레벨에 따라
  무한 체감 증가(전역 보너스와 동일 곡선 철학).
- 전역 월드 파워(#72)와 **이중 계산 없이** 독립 적용(서로 다른 효과·다른 지점).

---

## 2. 트랙별 로컬 보너스 (6종)

각 트랙 레벨 → 해당 시스템 소비 지점에서 적용. **중앙화**: 단일
`FMasteryFormula::GetLocalBonus(track, level)`(+서버 미러)가 값을 산출.

| 트랙 | 로컬 보너스 (자기 시스템 심화) | 적용 지점 |
| --- | --- | --- |
| **전투** | 처치 골드·EXP 보상 +X% | 처치 보상(IdleMonster/reward) |
| **장비** | 강화 골드 비용 **-X%** (클램프 최대 50%) | 강화 비용(`GetEnhanceCost` 소비) |
| **심연** | 던전 보상 +X% | 던전 보상(TryRunDungeon) |
| **룬** | 룬 코어 효과 +X% (코어 배수 가산) | RefreshDerivedStats 룬 코어 |
| **야성** | 펫 보너스 +X% | 펫 보너스 적용 지점 |
| **탐험** | 퀘스트 보상(골드/EXP) +X% | 퀘스트 수령(ClaimQuest) |

**곡선**: 증가형 `bonus = k·ln(1+level)` (전역과 동일 철학, 무한·체감). 장비
비용 절감은 `min(0.50, k·ln(1+level))` 클램프(공짜 강화 방지).

> **강화는 성공률(RNG·parity 민감) 대신 골드 비용 절감으로** — 깔끔한 곱,
> parity 리스크 회피, 여전히 "장비 시스템 심화" 충족.

---

## 3. 적용 & 이중 계산 방지

- 로컬 보너스는 **전역 월드 파워(#72)와 다른 효과·다른 지점**에 적용 → 이중 없음.
- 각 보너스는 **단일 적용 지점**(#72/#73 EXP 이중적용 교훈). 예: 전투 보상 보너스는
  처치 보상 1회, 강화 절감은 비용 계산 1회.
- 룬 코어 가산은 RefreshDerivedStats 룬 코어 곱 항에 합류(기존 룬/마스터리 코어와
  별도 항, 곱 폭주 없이 코어 배수에 가산).

## 4. 통합 지점 (구현 개요)

| 파트 | 작업 |
| --- | --- |
| character (메인) | `FMasteryFormula::GetLocalBonus(track, level)` + `UMasteryService::GetLocalBonus(track)`, 6개 소비 지점 적용(처치보상/강화비용/던전보상/룬코어/펫보너스/퀘스트보상), 클라 Automation. **세이브 변경 없음**(트랙 XP는 #72에 이미 저장) |
| backend | `mastery.ts`에 `localBonus(track, level)` 미러 + vitest parity |
| designer | 숙련 패널에 트랙별 **현재 로컬 보너스 표시** + ko/en 툴팁 |
| balance | 6종 계수 근거 + 강화 절감 클램프 + 무한 비폭주 문서 |
| qa | 보너스 단조 증가·강화 절감 클램프·각 시스템 적용·회귀 시나리오 + parity |

**세이브: 변경 없음**(SaveVersion 14 유지 — 로컬 보너스는 기존 트랙 레벨에서 파생).

## 5. 스코프

**In Scope (V1.5):** 6 트랙 로컬 보너스 1종씩(중앙화 공식), 6 소비 지점 적용,
숙련 패널 표시, 서버 미러+parity, balance 문서, 5-team.

**Out of Scope (후속):** 트랙당 2종 이상 로컬 보너스, 강화 성공률 보정(RNG),
던전 입장 횟수 +, 로컬 보너스 전용 UI 연출.
