# Backend V1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** PR #2 백엔드 V1 범위의 Fastify 서버, DB 마이그레이션, Docker 통합, API 문서, ADR을 한 번에 제공한다.

**Architecture:** Fastify 앱은 `server/src/server.ts`에서 코어 플러그인과 도메인 라우트를 조립한다. 도메인별 서비스는 repository 인터페이스를 받아 테스트 가능하게 두고, 실제 repository는 PostgreSQL과 Redis를 사용한다.

**Tech Stack:** Node.js 22, TypeScript 5.5, Fastify 5, Drizzle ORM, PostgreSQL 16, Redis 7, vitest, pino, bcrypt, JWT RS256.
