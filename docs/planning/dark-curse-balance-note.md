# Dark 저주(Curse) — 밸런스 노트 (PR #84)

> PM/Claude 직접 작성(Codex 한도 대체). #66 Dark + #30 상태이상 — 저주 = 받는 피해 증폭 디버프.

## 1. 저주 효과 / 수치
- **저주 = 받는 피해 증폭**: 대상이 저주 활성 동안 받는 피해 ×(1 + Magnitude). DoT(Poison/Burn)·슬로우(Freeze)와 다른 범주. 활성 중 최대 Magnitude 사용(스택 없음).
- 부여 스킬(Dark 속성, 신규 트리 없음):
  - **shadow_stab**(Thief): 지속 4초 / 증폭 **+20%**
  - **void_call**(Summoner): 지속 3초 / 증폭 **+15%**

## 2. 밸런스 / 인플레 점검
- 저주는 **자동전투 처치 가속**(받는 피해 증폭)이며 Dark 속성(Thief/Summoner) 빌드에 전투 정체성 부여.
- 증폭은 보수적(+15~20%, 지속 3~4초) — 상시 유지 아님(스킬 쿨다운/적중 조건). 영구 성장 아님.
- Holy↔Dark 상성(#66 ×1.5)과 별개 축(저주는 받는 피해 증폭, 약점은 가하는 피해). 이중 아님.
- balance-sim 코어 환생 페이싱 무관: median **5.328h**(inside-target) 무변동(저주는 전투 런타임, 사냥 효율 소폭 향상은 사냥터 진행에 흡수).

## 3. 비고
- **세이브 변경 없음**(전투 런타임 상태). 기존 Poison/Burn/Freeze 회귀 없음.
- 후속: 저주 스택, 추가 Dark 디버프, Dark 전용 스킬 트리.
