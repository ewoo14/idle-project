# 챕터5 밸런스 노트 (PR #80)

- 약점만 41~50 추가(Dark 가중) — 전투 상성(약점 ×1.5/저항 등)은 기존 공식 그대로, 밸런스 영향은 속성 매칭 변동뿐(스케일 불변).
- 5-5 미니보스/5-10 챕터보스는 기존 엘리트(X-5 ×3)/보스(X-10 ×8) 보상 일반화 재사용.
