## 30초 요약

- **장르**: 횡스크롤 도트 풍 방치형 RPG (메이플 스타일 영감, 자체 IP)
- **타깃**: PC (Steam) 1차, 추후 모바일
- **코어 루프**: 자동 사냥 → 골드/장비 획득 → 강화/진행 → 새 맵 해금 → 보스 → 환생
- **메타**: 환생 시스템, 펫, 길드(비동기), 시즌 패스
- **수익 모델 (검토 대상)**: 1차 출시 — 유료 게임 (DLC 가능). 운영형 콘텐츠는 1.0 이후 단계.
- **차별점**:
  - UE5 의 Niagara 이펙트로 도트 게임 대비 화려한 전투 연출
  - 환생 + 영구 보너스 + 시즌 메타로 장기 플레이 보장
  - 서버 권위적 세이브로 치트 방지 + 클라우드 동기화
