# PR #60 Codex Balance Notes

## Scope

- Added an eight-class balance snapshot to `tools/balance-sim`.
- Tuned Berserker and Summoner growth/skill coefficients in client and server
  mirrors.
- Added Vitest coverage for class balance bands, role exceptions, and formula
  anchors.
- Updated the balance philosophy with PR #60 guardrails.

## Formula Changes

### Berserker

- STR level bonus: `1.8 -> 1.5`.
- LUK level bonus: `0.65 -> 0.7`.
- Active damage coefficients:
  - `rage_cleave`: `2.8 -> 2.35`.
  - `blood_surge`: `1.9 -> 1.65`.
  - `savage_leap`: `2.4 -> 2.05`.

This keeps Berserker as a physical DPS class with better crit pressure, but
removes the previous strict-better STR curve over Warrior.

### Summoner

- INT level bonus: `1.7 -> 1.45`.
- CON level bonus: `0.3 -> 0.52`.
- Active damage coefficients:
  - `spirit_bolt`: `2.2 -> 1.9`.
  - `familiar_swarm`: `1.6 -> 1.45`.
  - `void_call`: `2.3 -> 2.0`.

This keeps Summoner on the magic route with status/AoE identity, but removes the
previous strict-better INT curve over Mage.
