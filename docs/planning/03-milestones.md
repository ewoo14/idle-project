# 마일스톤 — M0 ~ M6

각 마일스톤은 1~3개의 PR 슬라이스로 구성되며, 모든 PR 은 [`멀티 에이전트 워크플로우`](../workflow/01-pm-codex-claude-loop.md) 를 따릅니다.

---

## M0 — 기초 (Foundation)
**목표**: 리포 / 워크플로우 / 도구 체계 확립. 코드 산출물 없음.

| PR | 내용 |
| --- | --- |
| **PR #1 (본 PR)** | PM 기획서, 워크플로우, 에이전트 명세, CI 스켈레톤, infra 정의, README, 리포 정책 |

**완료 기준 (DoD)**:
- 모든 기획/워크플로우 문서가 한글로 작성되고 머지됨
- CI 가 빈 상태에서도 통과
- Codex 호출 헬퍼 스크립트가 동작 (1회 dry-run)
- 메모리 / CLAUDE.md 에 워크플로우 명시

---

## 변경 관리

- 마일스톤 범위/순서 변경은 PM 결정 → 본 문서 갱신 → PR 로 머지.
- 신규 슬라이스 추가 시 번호 규칙: `PR #N` 으로 순차 부여.
