# 마스터리 로컬 보너스 V2 (트랙당 2종) — 밸런스 노트 (PR #82)

> PM/Claude 직접 작성(Codex 한도 대체). #74 로컬 보너스 1종 위에 2종째 추가.

## 1. 2종 효과 / 곡선
- `GetLocalBonus2(track, level) = 0.01 × ln(1+level)` (Equipment/Beast 0.5 클램프). 심연은 정수 `GetAbyssBonusEntries = floor(level/50)`, 상한 +3.
- 1종과 비중복(다른 효과·다른 지점), 단일 적용.

| 트랙 | 1종(#74) | 2종(#82) |
| --- | --- | --- |
| 전투 | 처치 EXP +% | 처치 골드 +% |
| 장비 | 강화 골드비용 절감 | 큐브(재설정/등급) 골드 가격 절감(클램프 0.5) |
| 심연 | 던전 보상 +% | 던전 일일 입장 **+N**(floor(Lv/50), ≤+3) |
| 룬 | 룬 코어 가산 | 분해 에센스 +% |
| 야성 | 펫 보너스 +% | 펫 먹이 골드비용 절감(클램프 0.5) |
| 탐험 | 퀘스트 보상 +% | 오프라인 보상 +% |

## 2. 무한 비폭주 / 페이싱
- 모든 % 2종은 로그 체감(엔드 Lv100 ≈ 0.046=4.6%), 비용 절감/먹이비용은 0.5 클램프 → 공짜화 방지.
- 던전 입장 +N은 상한 +3(일일 3→6회)로 고정 — 무한 증가 아님.
- balance-sim 코어 환생 페이싱 무관: median **5.328h**(inside-target) 무변동.
