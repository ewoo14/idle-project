> 본 명칭은 시안이며 추후 마케팅 검토 후 확정합니다.

## 1. 정체성

### 1.1 한 줄 컨셉
> *"손 떼고 봐도 즐거운 사이드뷰 애니메풍 방치 RPG — 잠시 자리를 비워도 영웅은 자라고 있다."*

> **스타일 변경 이력** (2026-05-25): 본 GDD 초안 (PR #1) 의 "횡스크롤 도트 풍" → **3D 스타일라이즈드 애니메 풍** 으로 전환 (PR #11). 메이플 한국/일본 RPG 분위기에 더 정합. 도트 사이즈 (8x96 px) / 픽셀 아트 산출 표준은 폐기, 셀쉐이딩 3D + 표정 시스템 채택.

### 1.2 장르
- **주 장르**: 방치형(Idle) RPG
- **부 장르**: 사이드뷰 액션 RPG, 캐릭터 키우기, 자동 전투
- **참고 작품**: 메이플스토리 키우기, AFK Arena, Idle Heroes, Cookie Run: Kingdom (메타 진행), Octopath Traveler (사이드뷰 3D)

### 1.3 타깃
- **1차 플랫폼**: Windows PC (Steam 출시 목표)
- **2차**: macOS, 모바일 (iOS/Android) — 1.0 이후 검토
- **연령**: 12세 이상 (메이플 / 일본 애니메 톤)
- **세션 길이**: 짧음 (5~15분) × 자주 (하루 3~5회)

### 1.4 차별점
1. **3D 셀쉐이딩 + 풍부한 표정** — VRoid 기반 미소년/미소녀 캐릭터, ARKit blend shape 표정 시스템 (전투/피격/사망/레벨업 별 자연 반응)
2. **환생 메타가 강함** — 5~10시간 마다 한 사이클이 끝나고, 영구 보너스/스킨/스토리가 잠금 해제
3. **서버 권위적 세이브** — 클라우드 동기화 + 치트 방지 + 디바이스 이동
4. **자체 오리지널 IP** — 메이플 IP 미사용 (법적 안전 + 자체 세계관 자산)

### 1.5 비즈니스 가설 (1차 출시)
- 모델: **유료 게임 (예: 9,900~14,900원)** + 추후 코스메틱 DLC.
- 운영형 패스/가챠는 1.0 이후 검토. MVP 는 단일 결제로 끝나는 형태.

## 2. 코어 루프

```
[자동 사냥]
   ↓ 골드, 경험치, 장비, 자원
[성장/강화]
   ↓ 더 강한 캐릭터
[새 맵 도전]
   ↓ 보스 격파
[지역 해금 + 스토리]
   ↓ 일정 진행 후
[환생] → 영구 보너스 → 처음부터 더 강하게
```

### 2.1 1차 루프 (5~10분)
1. 게임 시작 → 자동 전투 진행 중 (마지막 사냥터)
2. 누적된 골드/장비 수령
3. 장비 강화, 스킬 분배, 펫 먹이주기
4. 새 맵으로 이동 (가능하면) → 다시 자동 전투

### 2.2 2차 루프 (1~2시간)
- 보스 격파 → 신규 지역 해금 → 스토리 진행
- 일일 미션 수행

### 2.3 3차 루프 (5~10시간)
- 환생(전직) → 보너스 포인트 영구 적립 → 스토리 다음 챕터

### 2.4 오프라인 보상
- 마지막 사냥터의 시간당 골드/경험치/장비를 **최대 12시간** 까지 적립.
- 12시간 초과분은 가중치 0.3 으로 누적 (장기 이탈 방지 + 복귀 보상 균형).
- 환생 횟수에 따라 최대 시간 +1h, 가중치 +0.05 보너스.

## 3. 캐릭터 시스템

### 3.1 직업 (Class)

MVP 는 **3직업** 으로 출발, 1.0 까지 5직업 확장.

| 직업 | 컨셉 | 주 능력치 | 플레이 스타일 |
| --- | --- | --- | --- |
| **전사 (Warrior)** | 근접 광검사 | STR · CON | 높은 체력, 광역 공격 |
| **마법사 (Mage)** | 원소 술사 | INT · WIS | 폭발적 마법 DPS, 종이 방어 |
| **궁수 (Archer)** | 정밀 사격수 | DEX · LUK | 크리티컬 특화, 중거리 |
| **도적 (Thief)** *(1.0)* | 그림자 단검 | DEX · LUK | 회피 + 후방 공격 |
| **힐러 (Cleric)** *(1.0)* | 빛의 사제* | WIS · INT | 자가 회복 + 버프 |

*힐러는 자동 전투에서 자가 힐 / 솔로 생존 특화. 멀티는 길드 컨텐츠에 사용.

### 3.2 능력치 (Stats)

#### 1차 능력치 (캐릭터 시트에 노출)
- **STR (힘)** — 물리 공격력
- **DEX (민첩)** — 명중, 공격 속도, 회피 일부
- **INT (지능)** — 마법 공격력, MP
- **WIS (지혜)** — MP 회복, 마법 저항
- **CON (체력)** — HP, 물리 방어
- **LUK (운)** — 크리티컬 확률, 드롭률 일부

#### 2차 능력치 (전투에 직접 사용)
- HP, MP
- 물리 공격력, 마법 공격력
- 물리 방어, 마법 방어
- 공격 속도, 이동 속도
- 명중률, 회피율
- 크리티컬 확률, 크리티컬 데미지
- 드롭률, 골드 획득량, 경험치 획득량

#### 능력치 공식 (초안)
```
물리 공격력 = STR × 2 + 무기 ATK + 강화 보너스
HP          = CON × 10 + 레벨 × 20 + 장비 보너스
크리티컬률  = LUK × 0.2% + 장비 옵션 합
```

상세 공식은 [`05-balance-philosophy.md`](05-balance-philosophy.md) 에서 정밀화.

### 3.3 레벨 & 경험치
- 레벨 1 ~ 200 (환생 전 캡)
- 경험치 곡선: `EXP(n) = round(50 × n^1.7 + 100 × n)` (초안)
- 레벨업 시 자유 분배 가능한 능력치 포인트 5점 + 자동 분배 옵션

### 3.4 스킬

각 직업별:
- **액티브 스킬 4종** (자동 발동, 쿨다운 기반)
- **패시브 스킬 2종** (영구 효과)
- **궁극기 1종** (보스전 전용, 게이지 누적식)

스킬 트리는 환생 단계까지 부분 잠금. 환생 시 일부 슬롯 영구 해금.

## 4. 장비 시스템

### 4.1 슬롯 (총 8슬롯)
- 무기 (1슬롯)
- 모자 / 상의 / 하의 / 신발 / 장갑 (5슬롯)
- 망토 (1슬롯)
- 액세서리 (1슬롯, 1.0 에서 2슬롯)

### 4.2 등급 (Rarity)

| 등급 | 색상 | 옵션 줄 수 | 강화 한계 |
| --- | --- | --- | --- |
| Common (일반) | 회색 | 1줄 | +5 |
| Uncommon (고급) | 초록 | 2줄 | +8 |
| Rare (희귀) | 파랑 | 3줄 | +10 |
| Epic (영웅) | 보라 | 3줄 + 세트옵션 1 | +12 |
| Legendary (전설) | 주황 | 4줄 + 세트옵션 2 | +15 |
| Mythic (신화) | 무지개 | 5줄 + 고유 효과 | +15, 진화 가능 |

### 4.3 강화 (Enhancement)

> 자세한 성공률 / 자원 소비량 / 천장은 [`05-balance-philosophy.md §2.3`](05-balance-philosophy.md) 단일 출처. 본 절은 개요만.

- **+0 ~ +5**: 100% 성공 (입문)
- **+6 ~ +10**: 90% 성공 (실패 시 단계 유지, 자원 소모)
- **+11 ~ +15**: 확률 강화 (70% → 30% 점진 감소, 실패 시 단계 -1~-3 + 신비석 추가 소모)
- 강화 재료: 강화석(공용), 신비석(고등급 전용)
- 천장: +11 이상 25회, +13 이상 10회 연속 실패 시 자동 성공

### 4.4 잠재 옵션 (Potential)
- 장비 옵션 줄에 랜덤 효과 부여 (예: "크리티컬 확률 +3%", "골드 획득량 +12%")
- 재감정 가능 (자원 소모)

### 4.5 세트 효과
- Epic 이상 동일 세트 2/4/6 부위 착용 시 추가 효과
- 첫 세트: "달빛 수호자" (2부위 — HP +10%, 4부위 — 모든 능력치 +5%, 6부위 — 궁극기 쿨다운 -10%)

## 6. 메타 진행

### 6.1 환생 (Rebirth)
- 조건: 챕터 1 보스 격파 + 레벨 100 도달
- 보상:
  - 영구 보너스 포인트 (모든 능력치 +1 ~ +5)
  - 환생 전용 스킬 슬롯 해금
  - 환생 횟수 ×N — 스토리/스킨 잠금 해제
- 초기화 대상: 캐릭터 레벨, 장비(일부 보존 가능), 골드(일부 보존)

### 6.2 펫
- 5종 (MVP 2종) — 강아지(골드+), 새(드롭+), 햄스터(경험치+), 슬라임(체력+), 여우(크리+)
- 먹이로 성장, 등급 진화
- 동시 1마리 장착, 1.0 에서 2~3마리

### 6.3 길드 (1.0)
- 비동기 협력: 길드 출석, 길드 던전 (비동기 보스 — 전체 누적 데미지)
- 시즌제 (4주)
- 채팅은 1.0 이후

### 6.4 시즌 패스 (1.0)
- 30일 시즌, 무료 트랙 + 프리미엄 트랙
- 일일/주간 미션으로 포인트 적립

## 8. 스토리 (요약)

> 자세한 시나리오는 [`docs/planning/06-story-bible.md`](06-story-bible.md) (Codex 스토리 에이전트 작성 예정) 에서.

- **세계**: 차원이 흔들리는 판타지 대륙 **에테르넬리아**.
- **주인공**: 평범한 모험가, 차원 균열에 휘말리며 영웅의 운명을 깨달음.
- **막 구성**:
  - 1막: 마을의 변고 (튜토리얼 + 챕터 1)
  - 2막: 잊혀진 신전의 비밀 (1.0)
  - 3막: 차원 군주의 그림자 (운영)
- **분위기**: 메이플 스타일의 밝은 동화풍 + 후반부 어두운 미스터리.

## 부록 A — 용어집

- **자동 전투(Auto Battle)**: 플레이어 입력 없이 캐릭터가 자동으로 공격·이동·스킬 사용
- **오프라인 보상(Offline Reward)**: 게임이 꺼져있는 동안 누적된 진행
- **환생(Rebirth)**: 일정 진행 후 캐릭터를 초기화하고 영구 보너스를 얻는 메타 시스템
- **잠재(Potential)**: 장비에 부여되는 랜덤 추가 옵션
- **세트 효과(Set Bonus)**: 같은 세트 장비를 N부위 착용 시 추가 능력
- **챕터(Chapter)**: 스토리/맵의 묶음 단위, 보스로 마무리됨
